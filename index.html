WebSocket

Now there is a WebSocket running on "/websocket/:roomid"

REQUIRED: replace :roomnumber with any string you want to use to make your websockets available only by that room id. Leaving it blank will result in unsuccessful WebSocket connection.

use this to connect to it on your client side:

const socket = new WebSocket('wss://ganisproxyserver.onrender.com/websocket/');

To receive messages:

socket.addEventListener("message", msg => {});

To send messages:

socket.send("Message to the server and all other clients. ");

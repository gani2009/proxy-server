api.textyl.co

Use route "/api-lyrics/"

then put the name of your request at the end of the route

Example: /api-lyrics/the%20beatles%20norwegian%20wood

This means you can just use fetch() to get the json

WebSocket

Now there is a WebSocket running on "/" url. i.e. the home url

use this to connect to it on your client side:

const socket = new WebSocket('wss://ganisproxyserver.onrender.com');

To receive messages:

socket.addEventListener("message", msg => {});

To send messages:

socket.send("Message to the server and all other clients. ");
